Shinoa File System

The Shinoa File System works with your GameID, thats why

your gameid can't have symbols or any unicode characters.

Is not recomended to place scripts inside the ShinoaFileSystem.

Packages

Package Name Description
assets.[gameid].textures This is the package for Game Textures.
assets.[gameid].shaders A Package to hold GLSL Shaders.
assets.[gameid].models This is the package for your Models.
